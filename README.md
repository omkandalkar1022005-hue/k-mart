# 🛒 Kmart - Multilingual Online Mart Website

## 📋 Project Overview
A responsive e-commerce website with **3 languages** (Marathi, Hindi, English) built using HTML, CSS & JavaScript.

---

## ✨ Key Features

### 🌐 Languages
- ✅ Marathi (Default) | ✅ Hindi | ✅ English
- ✅ Auto-saves preference in localStorage

### 📱 Responsive Design
- ✅ Mobile, Tablet & Desktop friendly
- ✅ Fixed header with sticky navigation

### 🛍️ Pages
| Page | File | Features |
|------|------|----------|
| Home | `index.html` | Hero section, categories, featured products |
| New Products | `products.html` | Filters, sorting, pagination |
| Best Sellers | `bestsellers.html` | Rankings, sold counts |
| Offers | `offers.html` | Countdown timer, coupons |
| Contact | `contact.html` | Form, FAQ, map |
| About Us | `about.html` | Team, stats, story |
